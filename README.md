# 🚀 Desafio 30 Dias de JavaScript — Simulando Desafios Corporativos

## 📌 Sobre o Desafio

Este repositório documenta minha jornada de **30 dias resolvendo desafios em JavaScript**, com foco em **problemas inspirados em tarefas reais do ambiente corporativo**.

A proposta é exercitar lógica, boas práticas, legibilidade e aplicabilidade prática — como se eu estivesse atuando dentro de um time de produto em uma empresa de tecnologia.

---

## 🎯 Objetivos Estratégicos

- 🔍 **Aproximação do mundo real:** Resolver problemas que simulam demandas comuns do cotidiano de desenvolvedores front-end.
- 📈 **Domínio técnico:** Consolidar fundamentos de JavaScript moderno (ES6+), incluindo lógica, estrutura de dados e funções.
- 🧠 **Clareza e comunicação:** Escrever soluções claras e documentadas como se estivesse explicando para uma equipe.
- 💼 **Empregabilidade ativa:** Demonstrar proatividade, raciocínio lógico e capacidade de autogerenciamento — diferenciais buscados por empresas tech.

---

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **Node.js (para execução via terminal, se necessário)**
- **Markdown (documentação)**
